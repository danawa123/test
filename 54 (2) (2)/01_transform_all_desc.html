transform:변형할 때 사용하는 기능

transform은 변형시 모든 축이 객체의 가운데를 중심으로 함

이동 변형: translate(기본 단위 px)

이동 x and y : transform:translate(x축, y축)

이동 x 축 : transform:translatex (x축으로 이동가능)

이동 y 축 : transform:translatey (y축으로 이동가능)

크기 변형: scale(100% == 1 , 단위없음)

크기변형 x and y : transform:scale( 1(x축) , 1(y축) )

크기변형 x : transform:scalex(0.5)

크기변형 y : transform:scaley(0.7)

회전 변형: rotate(45도 == 각도(deg))

회전변형 2d(평면) : transform:rotate(45deg)

회전변형 x축(3d) : transform:rotatex(60deg)

회전변형 y축(3d) : transform:rotatey(50deg)

회전변형 3d축 : transform:rotate3d(x축, y축, z축, 각도)

기울 이기 : skew(x축 각도(deg) , y축 각도(deg) )

기울이기 변형:transform:skew(x축 각도, y축 각도)

기울이기 변형 x축: transform:skewx( x축 각도)

기울이기 변형 y축: transform:skewy( y축 각도)

모든 transform은 각각 x축과 y축을 구분하여 사용할 수 있음
