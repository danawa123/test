유효성 검사: required="required"
  1. 필수정보 입력공간이 빈칸이 아닌지 확인하는 작업
  2. 문법에 맞는지 확인하기 (문자, 숫자, 한글, 영문인지 체크)
  3. 문법에 맞지 않거나 빈칸 일 경우, 알림창이나 경고창 띄우기
  4. 정교하게 문자열을 비교하거나 숫자를 비교할때, 자바스크립트나 제이쿼리 사용
  5. 위에 작업이 복잡하지 않을때는 html5만 가지고 충분히 해결가능함!

select 태그 VS datalist 태그

  1. 공통으로 option이란 태그 사용
  2. select 태그의 경우 하단의 메뉴를 재선택 할 수 있음
  3. datalist 의 경우 input공간에 인식이 되면 재선택이 어려움!
  4. datalist 의 경우 화면 어디에 만들어도 input 어디서든 호출이 가능함!
  5. select의 경우 화면 어디에 만들어도 호출하는 것이 아니라 계속해서 만들어서 사용해야 함!
  6. datalist는 한번 만들어 놓고 어디든 재 사용가능
  7. select 태그는 계속 만들어서 사용!
