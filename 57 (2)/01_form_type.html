html5에서 사용하는 form

form에 대한 중요 속성들

  1. action : 주소를 연결하기 위하여 사용 (주소줄을 이용하여 control 페이지나 db로 넘겨줌)
  2. method : get, post
  3. get: 주소줄에 넘어가는 정보를 보여줘도 될 때 (검색이 가능한 페이지)
  4. post : 주소줄에 넘어가는 정보를 보여주지 않을 때 (로그인, 회원가입)

  • type="checkbox"

html5에서 type사용에 대한 목적

1) type 정확한 "목적에 맞춰서 작업하기" 위하여 type 사용
2) html5에서는 반응형 페이지 제작이 가능함
반응형 페이지, 다양한 모바일 기기에 웹 페이지가 출력될 수 있도록 하는 작업
3) 각종 디바이스 기기들에서 키패드가 적절하게 뜨도록 만드는 역할을 담당함!
4) 대다수의 표준브라우저에서 사용가능
5) 익스플로러에서는 표준브라우저 버전이지만 적용불가능한 type이 많음!
중요) type은 에러가 발생 할 확률이 높음 (지원되지 않는 브라우저들의 문제)
