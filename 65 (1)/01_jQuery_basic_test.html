첫번째 문법의 장점과 단점

제이쿼리의 문서가 준비가 되면 기능 구현해 주세요.

  • 장점1: 문서가 준비가 되면 기능구현시 가장 안전함, 복잡한 기능도 안전하게 구현할 수 있음!
  • 장점2: $셀렉터를 사용하지 않고, jQuery를 사용하기 때문에 php와 충돌의 우려가 없음!
  • 단점: 문서가 준비되지 않으면 화면에 아무것도 나타나지 않고, 계속해서 로딩이 걸림, 페이지를 준비해야 하기 때문에 무한 loop 현상에 빠져버림!
  • 정리: 문서가 준비되지 않으면 무한으로 로딩되기 때문에 이럴 경우, 세번째나 네번째 문법으로 해결한다.
    장점과 단점이 확실하기 때문에 서로 해결이 가능함!

문제3: p태그 만들고 script 작업하기

  1. body: p태그 만들기 작성
  2. script: outline 1px solid blue 적용하기
  3. 세번째 문법 사용하기( 셀렉터를 이용한.. )

세번째 문법의 장점과 단점

셀렉터를 이용하여 기능구현 해주세요

  • 장점1: 대상을 선택할때 jQuery가 아니라 $(셀렉터)로 선택하기 때문에 대상선택이 매우 빠름
  • 중요) $는 선택할때 사용하는 기호, script가 아니어도 대상 선택시 사용할수 있는 기호로 대상을 선택할때 만 사용한다.
  • 장점2: 문서를 준비하지 않고, 기능을 구현하기 때문에 기다릴 필요가 없음, 읽어주는 속도가 매우 빠름
  • 단점1: php에서는 선택자로 $만 사용가능함, php 문서와 jQuery 문서를 함께 사용할 경우, 특히 대상을 $로 잡을 경우 , 서로의 선택자로 인해 오류가 생김!
  • 단점2: 브라우저를 활성화 하자마자 기능을 구현하기 때문에 기능이나 문서가 깨질 우려가 높음, 화면에 아무것도 출력되지 않거나, 이벤트 기능들(hover, focus)이 작동하지 않을 수 있음
  • 정리: 장점은 처리속도가 매우 빠름, 단점은 깨질 우려가 높음, 페이지의 오류 생성 확률이 매우 높음!
